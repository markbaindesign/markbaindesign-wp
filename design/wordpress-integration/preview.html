Links

Bain links are always underlined blue. On hover, color shifts to ink and the underline thickens. Visited links are classic browser purple — that's deliberate. External links use the ↗ glyph.

Code

Inline wp_enqueue_style() calls live in a single hook so dependency order is explicit:

add_action( 'wp_enqueue_scripts', function () {
    wp_enqueue_style( 'bain-fonts',  $google_fonts_url );
    wp_enqueue_style( 'bain-tokens', get_theme_file_uri( 'assets/css/tokens.css' ), [ 'bain-fonts' ] );
    wp_enqueue_style( 'bain-base',   get_theme_file_uri( 'assets/css/base.css' ),   [ 'bain-tokens' ] );
    wp_enqueue_style( 'bain-theme',  get_stylesheet_uri(),                          [ 'bain-base' ] );
} );
